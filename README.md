# 🎓 College Inventory Management System  

> A sleek and responsive web-based inventory management solution designed for colleges and educational institutions.

Manage assets, track department allocations, handle requests, and organize resources — all from one modern dashboard.

---

## 🚀 Overview

The **College Inventory Management System** is a frontend-based project built using **HTML, CSS, and JavaScript** that helps colleges efficiently manage inventory and institutional resources.

The system allows users to:
- Manage inventory items
- Track department allocations
- Handle item requests
- Generate reports
- Maintain secure backups

---

## ✨ Features

- 📦 Inventory Management System
- 🏢 Department-wise Allocation Tracking
- 📝 Item Request Management
- 🔐 Admin & User Authentication
- 💾 Auto Backup & Restore
- 📊 Export to Excel & PDF
- 🔍 Search & Filter Functionality
- ⚡ Quick Add Common Items
- 📱 Fully Responsive Modern UI

---

## 🛠️ Technologies Used

- **HTML5** → Structure of the website
- **CSS3** → Styling and responsive design
- **JavaScript (Vanilla JS)** → Functionality and interactivity
- **Local Storage API** → Data persistence & backup
